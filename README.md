#  Web Works 3D
Random Three.js models and other web 3d experiments.

[Live site]()

## How to use
Clone the repo

```
git clone ...
```

Install packages:

```
npm install
```
Run in local server:

```
npm start
```

## Framework and language
* Tailwind CSS
* JavaScript / Vue.js
* Vite.js

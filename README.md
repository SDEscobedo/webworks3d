#  Web Works 3D
Random Three.js models and other web 3d experiments.

[Live site](https://webworks3d.netlify.app/)

## How to use
Clone the repo

```
git clone https://github.com/SDEscobedo/webworks3d.git
```

Install packages:

```
npm install
```
Run in local server:

```
npm start
```

## Framework and language
* Tailwind CSS
* JavaScript / Vue.js
* Vite.js
